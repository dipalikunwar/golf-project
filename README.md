# ⛳ Golf Web App

A modern and interactive **golf-themed web application** built using HTML, CSS, and JavaScript.
This project includes multiple pages like login, dashboard, and homepage to simulate a real-world web experience.

---

## 🚀 Features

* 🔐 Login Page (User authentication UI)
* 📊 Dashboard Page
* 🏠 Home Page (Landing page)
* 🎨 Clean and responsive UI
* ⚡ Smooth user interaction using JavaScript

---

## 🛠️ Tech Stack

* **HTML5** – Structure
* **CSS3** – Styling
* **JavaScript (app.js)** – Functionality

---

## 📂 Project Structure

```
GOLF-APP/
│── index.html        # Home page
│── login.html        # Login page
│── dashboard.html    # Dashboard page
│── style.css         # Styling
│── app.js            # JavaScript logic
│── golf.png          # Image asset
```
